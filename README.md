# homestay-system🎂
基于SpringBoot+Vue的民宿预定系统

#### 介绍🌞
此次设计将要模拟完成建立一个民宿管理系统系统，其目的是在能够降低管理成本的前提下，实现对民宿管理的信息化，提高对民宿的管理效率以及提升民宿的服务质量。

#### 安装教程🌞

1. 运行环境准备mysql8 + java8 + node14.16.1

2. 配置maven路径，加载依赖

3. 运行sql文件，确保application.yml的数据库名称和账号密码是数据库所在主机的账号密码
